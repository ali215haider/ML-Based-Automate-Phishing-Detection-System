## Library Names with Single Line Descriptions

Here's a comprehensive list of all libraries used in your Automate Phishing Detection System with brief descriptions:

### **Web Framework & Core**
- **Flask** - Lightweight Python web framework for building web applications
- **Flask-SQLAlchemy** - Flask extension that adds SQLAlchemy ORM support
- **Flask-Login** - User session management and authentication for Flask
- **Flask-WTF** - Integration of Flask and WTForms for form handling
- **Flask-Migrate** - Database migration support for Flask applications
- **Flask-Dance** - OAuth consumer extension for Flask applications
- **Flask-Mail** - Email sending capabilities for Flask applications
- **Werkzeug** - WSGI utility library that powers Flask
- **Gunicorn** - Python WSGI HTTP Server for UNIX systems

### **Machine Learning & Data Analysis**
- **scikit-learn** - Machine learning library with classification and regression algorithms
- **pandas** - Data manipulation and analysis library with DataFrame structures
- **numpy** - Fundamental package for scientific computing with arrays
- **joblib** - Lightweight pipelining and parallel computing library

### **Web Scraping & HTML Processing**
- **requests** - HTTP library for making web requests and API calls
- **beautifulsoup4** - HTML and XML parsing library for web scraping
- **lxml** - Fast XML and HTML parser with XPath support
- **html5lib** - Pure-Python HTML5 parser based on WHATWG specification
- **bleach** - HTML sanitization library to prevent XSS attacks

### **Security & Cryptography**
- **cryptography** - Cryptographic recipes and primitives for Python
- **PyOpenSSL** - Python wrapper around OpenSSL library for SSL/TLS
- **ssl** - SSL/TLS wrapper for socket objects (built-in module)
- **PyJWT** - JSON Web Token implementation for Python
- **oauthlib** - Generic OAuth request-signing logic implementation

### **Network & Domain Analysis**
- **urllib3** - HTTP client library with connection pooling and SSL/TLS verification
- **whois** - Python library for retrieving WHOIS information about domains

- **tldextract** - Library to extract domain components from URLs
